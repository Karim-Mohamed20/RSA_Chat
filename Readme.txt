## How to run
1- open terminal and run command "python server.py"
2- open another terminal and run command "python client.py"
3- from client terminal, start chatting and send message , it will encrypted and server will receive it and dycrypt the message then server will be ready to send message, it will encrypted and client will recieve it and decrypt the message and client will be ready to send a message and so on
4- for breaking RSA, open terminal and run command "python RSA_break.py", you can first generate public key and n and cipher text by running command "python test_Time.py" and use them as input in RSA_break file
